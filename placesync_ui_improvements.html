Upgrade Theme.java — fonts, shadows, spacing tokens

LoginPanel — dark teal background + floating card
Paint SIDEBAR_BG as the panel background. Make the card use paintComponent for a custom shadow. Replace the flat GridLayout toggle with a pill toggle using a shared ButtonGroup and CardLayout for the active indicator. ↗

Sidebar — add mini labels below icons, darken bg
Change sidebar width from 60px to 72px. Each nav item gets a 2-line JPanel (icon on top, 9px label below). Active item gets a rounded highlight rect via paintComponent with SIDEBAR_BG.brighter(). ↗
4
Tables — status badge cell renderer + row hover
Create a StatusCellRenderer extends DefaultTableCellRenderer that paints a colored rounded rectangle (green/amber/red) inside the cell. Add a MouseMotionListener to track hovered row and repaint with a subtle background tint. ↗

Card.java — fix the hardcoded "+12% from last month" trend
The trend text is currently identical on every card. Add a setTrend(String text, boolean positive) method. Pass the real trend string from each dashboard. Color it ACCENT_EMERALD (up) or red (down). ↗
6
Topbar — add user avatar + breadcrumb context
Extend Topbar.java to show a circular avatar with initials (painted via fillOval), the logged-in user's name, and the current panel name as a breadcrumb on the left. ↗
